HTML Computer Code

HTML Computer code is used to write computer or programming language codes on the web page. kbd tag, samp tag, and code tag change the font style of the text and make it smaller than actual text which is suitable to write computer code.

HTML Keyboard Input

  • HTML Keyboard input is used to write keyboard inputs like (Ctrl+C, F5 etc.) on the web page.

  • HTML Keyboard input is defined by <kbd> tag.

HTML Computer Output

  • HTML Computer output is used to write computer output like (Error!, Access denied, File Not Found etc.) on the web page.

  • HTML Computer output is defined by <samp> tag.

HTML Computer Code

  • HTML Computer code is used to write computer code like (#include, function() etc.) on the web page.

  • HTML Computer code is defined by <code> tag.

"The default font family of <kbd>, <samp>, <code> tag is monospace because that is the most appropriate to write computer code."

Example:
<kbd>Ctrl+C</kbd>
<samp>Error!</samp>
<code>function()</code>
